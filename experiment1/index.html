Experiment 2 - Vector Art, Animation & Interactivity

Description

This is an experiment based on class project references!

I'm exploring various interactive animations using p5.js, where users can engage with the artwork in a dynamic, fun, and interesting way.

Technical

In these experiments, I'm utilizing p5.js to create animations and interactive vector art. The experiment responds to user input in distinct ways clicking and moving the mouse around interacts with the piece.

Reflection

The challenges included conceptualizing engaging user interactions, and making sure it would be an interesting piece of art that users can enjoy looking at.
