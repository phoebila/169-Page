Experiment 7: Data Visualization & Networks

Description

The Mood Weaving Scarf is an interactive, generative artwork in p5.js that visualizes emotional states over time. Each day, users input their mood by selecting a button, which adds a new woven stripe to the digital scarf. The scarf grows over time, reflecting the userâ€™s emotional history through color and pattern.

Technical

This piece as mood input button that's stored in local storage and has a clear cache button to clear input. Theres a dictionary for the colors, a generative scarf algorithm and a pattern effect.

Reflection

Straightforward piece, wanted to create something simple to have more time for capstone project.
